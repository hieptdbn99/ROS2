Setup Instructions for Remote Control:

  1. Find the ROS2 PC's IP address: 
    ip addr show | grep inet
  2. Make sure firewall allows port 8765: 
    sudo ufw allow 8765
  3. Start your ROS2 node: 
    ros2 run my_robot_controller demo_node
  4. Enter the IP address above and click Connect
  5. Optional: Start turtle simulator to see the turtle move: 
    ros2 run turtlesim turtlesim_node
